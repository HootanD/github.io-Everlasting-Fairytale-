## 🆘 If Something Fails

### Script Hangs?
- Press `Ctrl+C` to exit
- Check specific step manually
- Read `TROUBLESHOOTING.txt`

### Authentication Fails?
```bash
gh auth logout
gh auth login
# Provide GitHub Personal Access Token (ghp_...)
```

### Secrets Not Set?
```bash
gh secret list  # Verify
gh secret set SECRET_NAME --body "value"  # Re-set
```

### Git Push Fails?
```bash
git status  # Check uncommitted changes
git log -1  # Verify last commit
git push origin main --force-with-lease  # Retry
```

### Workflows Not Triggering?
```bash
gh run list --branch main  # Check if any runs started
gh workflow list  # Verify workflows exist
gh workflow run ci-cd.yml --ref main  # Manual trigger
```

   ```bash
   docker tag docker.io/user/repo:commit-sha docker.io/user/repo:stable
   docker-compose -f docker-compose.prod.yml up -d app-stable
   ```
